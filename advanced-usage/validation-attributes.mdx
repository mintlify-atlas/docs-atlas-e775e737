---
title: Validation Attributes
description: Complete reference of all validation attributes available in Laravel Data
---

Laravel Data provides validation attributes that correspond to Laravel's validation rules. Use these attributes on data properties to automatically generate validation rules.

## Database

<AccordionGroup>
<Accordion title="Exists">
[Docs](https://laravel.com/docs/validation#rule-exists)

```php
#[Exists('users')]
public string $userId;

#[Exists(User::class)]
public string $userId;

#[Exists('users', 'email')]
public string $email;

#[Exists('users', 'email', connection: 'tenant')]
public string $email;

#[Exists('users', 'email', withoutTrashed: true)]
public string $email;
```
</Accordion>

<Accordion title="Unique">
[Docs](https://laravel.com/docs/validation#rule-unique)

```php
#[Unique('users')]
public string $email;

#[Unique(User::class)]
public string $email;

#[Unique('users', 'email')]
public string $email;

#[Unique('users', connection: 'tenant')]
public string $email;

#[Unique('users', withoutTrashed: true)]
public string $email;

#[Unique('users', ignore: 5)]
public string $email;

#[Unique('users', ignore: new AuthenticatedUserReference())]
public string $email;

#[Unique('posts', ignore: new RouteParameterReference('post'))]
public string $slug;
```
</Accordion>
</AccordionGroup>
