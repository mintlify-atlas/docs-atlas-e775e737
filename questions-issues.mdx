## Getting Help

Find yourself stuck using the package? Found a bug? Do you have general questions or suggestions for improving Laravel Data?

### GitHub Issues

Feel free to [create an issue on GitHub](https://github.com/spatie/laravel-data/issues). We'll try to address it as soon as possible.

## Security Vulnerabilities

<Warning>
  If you've found a bug regarding security, please email [freek@spatie.be](mailto:freek@spatie.be) instead of using the issue tracker.
</Warning>

We take security vulnerabilities seriously. Please do not disclose security issues publicly until they have been addressed by our team.

## Support Spatie

We invest a lot of resources into creating best in class open source packages. You can support us by:

- [Buying one of our paid products](https://spatie.be/open-source/support-us)
- Sending us a postcard from your hometown to our [contact address](https://spatie.be/about-us)
- Starring the repository on GitHub
- Sharing the package with your network

All received postcards are published on [our virtual postcard wall](https://spatie.be/open-source/postcards).
